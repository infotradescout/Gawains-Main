# MealScout

Repo: `infotradescout/MealScout`

## Purpose

MealScout is for restaurants, food trucks, menus, schedules, public profiles, onboarding, and affiliate tracking.

## Doctrine

- Restaurants/trucks only.
- No TradeScout contractor language.
- No fake menus.
- No invented schedules.
- No fake locations.
- Public profiles must be useful without deals.
- Every shared link while logged in should preserve affiliate attribution.
- Images supplied manually must land in the same storage path/shape as user uploads.

Add latest baseline and active lanes when MealScout work resumes.
